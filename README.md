This project for Portfolio Purpouse

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

- Supabase

- Prisma
- Socket.io
- Nodemailer
